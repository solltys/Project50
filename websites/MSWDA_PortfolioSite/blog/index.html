JavaScript Frameworks in 2020

It's 2020 and the JavaScript train doesn't look like its about to slow down any time soon. As JavaScript evolves, the frameworks that go with it keep popping up and changing at at an amazing pace. Within three months, a framework can go from something nobody's ever heard of to the hottest new thing, only to be forgotten as soon as the next big thing comes along.

Let's look at some of this year's most popular JavaScript frameworks:

  • Angular
  • Backbone.js
  • Ember
  • React
  • Vue

You should really take a look at the big five frameworks, and choose the one that best suits your needs. Personally, I'm using React quite a bit these days, as I find it provides just enough tools to get a project started without overwhelming me with built-in stuff. Here's how easy it is to create a simple React page:

ReactDOM.render(   <h1>Hello, world!</h1>,   document.getElementById('root') );

If using React, remember folks:
State is king!

Programming Languages: The Greats

The other day, a student asked me which were the most influential programming languages ever made. We came up with quite a list, although it was hard to shortlist them, since every language contributes one or more things to the field. Nevertheless, here it is, the list of the most influential programming languages - at least in my opinion!

Language Year Released Created by
FORTRAN 1957 John Backus & IBM
COBOL 1959 CODASYL
C 1972 Dennis Ritchie & Bell Labs
C++ 1985 Bjarne Stroustrup
Python 1990 Guido Van Rossum
Java 1995 James Gosling & Sun Microsystems
JavaScript 1995 Brendan Eich

What do C++, Python, Java and JavaScript have in common? They are all based on the syntax of C, along with dozens of other programming languages. This is a testament to the genius of the creator of the C programming language, the late Dennis Ritchie.
